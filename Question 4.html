Ordered List of Html Tags

  1. Html : The html tag represents the root of an HTML document.
  2. Head : The head element is a container for metadata (data about data) and is placed between the html tag and the body tag
  3. Meta : The meta tag defines metadata about an HTML document. Metadata is data (information) about data.
  4. Title : The title tag defines the title of the document. The title must be text-only, and it is shown in the browser's title bar or in the page's tab.
  5. Body : The body tag defines the document's body.
  6. a : The a tag defines a hyperlink, which is used to link from one page to another.
  7. h1 to h6 : The h1 to h6 tags are used to define HTML headings.
  8. div : The div tag defines a division or a section in an HTML document.
